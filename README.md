## REST Endpoints

Name                           | Method | Path
-------------------------------|--------|------------------
Retrieve tasks collection  | GET    | /tasks
Retrieve tasks with tag    | GET    | /tasks/*\<tag\>*
Create task                | POST   | /tasks
Update task                | PUT    | /tasks/*\<id\>*
Delete task                | DELETE | /tasks/*\<id\>*
